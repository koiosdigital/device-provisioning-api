Pass the `CloudflareBindings` as generics when instantiation `Hono`:

```ts
// src/index.ts
const app = new Hono<{ Bindings: CloudflareBindings }>();
```

### API reference

An OpenAPI document is exposed at `/swagger.json`. Import it into your preferred API client (Insomnia, Postman, Stoplight, etc.) to explore available routes and payload requirements.
